## 🔍 Project Breakdown

### ☁️ Cloud Resume
Deployed a personal resume as a static site on AWS — no servers, no frameworks, just cloud infrastructure.

**What I built:**
- Hosted the site on **S3** as a static website
- Distributed it globally via **CloudFront** with HTTPS
- Pointed a custom domain using **Route 53**

**What I learned:**
- How S3 bucket policies and static hosting work
- CDN concepts: edge caching, invalidations, origin behavior
- The difference between S3 URLs and CloudFront distributions
